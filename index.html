Architecture Logic

The Sentrix framework establishes a secure perimeter via the VPN. The Windows DC acts as the Single Source of Truth for identity. The Linux Nodes point to the DC for DNS queries, allowing them to resolve internal hostnames like app-server.sentrix.local. All system logs are shipped via Filebeat to the ELK Stack for real-time security auditing.
